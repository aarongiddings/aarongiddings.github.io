# Projects

## NBA Action Recognition Dataset

Building a curated dataset of short NBA broadcast clips labeled with common basketball actions. The dataset supports training and benchmarking action-recognition models for sports analytics research.

Includes:

- manual annotation workflows
- bounding-box generation
- temporal action segmentation
- dataset formatting for deep learning pipelines

---

## Basketball Object Detection with YOLO

Training object detection models to identify:

- players
- referees
- ball
- basket
- backboard

from broadcast video frames. These detections support downstream event classification models.

GitHub:  
https://github.com/aarongiddings

## Numerical Solvers for Burgers’ Equation

Implemented multiple numerical schemes for nonlinear PDE systems:

- finite difference solvers
- WENO methods
- TVD schemes
- PINN-based approaches

Compared stability, accuracy, and sampling strategies across solution methods.

---

## PINNs with Adaptive Sampling

Explored residual-based and gradient-based adaptive sampling strategies to improve neural-network solutions of PDEs.

Compared:

- grid sampling
- uniform sampling
- Halton sequences
- Hammersley sequences
- ASM I
- ASM II
